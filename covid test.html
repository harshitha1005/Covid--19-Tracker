COVID-19 TEST

VIRAL TEST

Viral test will only tell if you are infected now.
If you test negative,you can still get infected.

ANTIBODY TEST

It checks if you were infected with virus before.
It could be gone or you could still be contagious.

LAB TESTS

1)SWAB TEST-Swab is used to take sample from nose or throat.


2)TRACHEAL ASPIRATE TEST-Bronchoscope is put into mouth to reach lungs from where sample is collected.


3)BLOOD TEST-Blood sample is taken from vein in the arm.
